Tasks api documentation

Usage:

GET all tasks: /tasks

GET one task: /tasks/{task_id}

POST new task: /tasks (JSON body with all fields)

PUT update one task: /tasks/{task_id} (JSON body, at least one field)

DELETE delete one task: /tasks/{task_id}


Task fields:

{
  _id: string // autogenerated UUID
  title: string, // MAX 50 chars
  category: string, // MAX 50 chars
  completed: boolean,
  dueHour: string // format HH:MM
}
